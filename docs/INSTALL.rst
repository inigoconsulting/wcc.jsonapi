wcc.jsonapi Installation
------------------------

To install wcc.jsonapi using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.jsonapi`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.jsonapi

* Re-run buildout, e.g. with:

    $ ./bin/buildout
